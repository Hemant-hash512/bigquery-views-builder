# bigquery-views-builder
clone to get an easy setup for version control and automatic builds of your bigquery views. Add your view DDLs under the folders that represent the respective dataset the view belongs to. Set up a trigger in cloud build that listens to changes of views/** in the branch of your choice.

Read more at [https://robertsahlin.com/automatic-builds-and-version-control-of-your-bigquery-views/](https://robertsahlin.com/automatic-builds-and-version-control-of-your-bigquery-views/)
